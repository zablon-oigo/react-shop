# shop
## Tech Stack

**Client:** React, TailwindCSS
## Run Locally

Clone the project

```bash
  git clone git@github.com:zablon-oigo/react-shop.git
```

Go to the project directory

```bash
  cd react-shop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
